Introduction to HTML Tables

Tables are used to arrange data in rows and columns. HTML provides tags like <table>, <tr>, <td>, and <th> for creating tables.

Rows and Columns ‚äû

Rows are created with <tr> and columns with <td>.

<table> Tag

The <table> tag defines the start and end of a table.

<td> - Table Data Cell

The <td> tag defines a single data cell inside a row.

Adding Borders to a Table

Borders make the table clearer. You can use the border attribute in the <table> tag.

<tr> - Table Row

The <tr> tag defines a complete row in a table.

<th> - Table Header Cell

The <th> tag defines a header cell (bold & centered by default).

Table Header, Body, and Footer

HTML allows dividing tables into <thead>, <tbody>, and <tfoot> sections.

Quick Review of Tables

The colspan attribute allows a cell to span across multiple columns.

Understanding Rowspan
