About This Project

This project is created as part of a Hackathon task, using only pure HTML (no CSS, no JavaScript). The goal is to provide an easy-to-access resource for anyone who wants to learn the basics of HTML step by step.

How to Use

  • Click on any lecture link to read the content.
  • Each lecture page contains definitions, examples, and explanations.
  • You can always return to this homepage using the "Back to Homepage" link provided on every page.
