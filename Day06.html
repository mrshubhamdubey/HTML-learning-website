Media Tag üé¨

The Media Tags in HTML allow us to integrate audio and video content into webpages. They provide controls like play, pause, and volume, making websites interactive.

Integrating Audio Files üîä 

<audio src="audio/song.mp3" controls></audio>

controls: attribute adds play/pause/volume options.

Audio: Proper Formats

  • MP3 (Most common, supported in all browsers)
  • WAV (High quality, but large size)
  • OGG (Open-source format)

Why Proper Formats are Needed? Because not all browsers support every file format. MP3 ensures maximum compatibility.

Integrating Video Files üéûÔ∏è

<video src="video/sample.mp4" controls width="400"></video>

controls: lets user control video playback.

Video: Proper Formats

  • MP4 (Most widely supported)
  • WebM (Open-source format)
  • OGG (Some browsers support)

Embedding YouTube Videos [‚ñ∂Youtube]

YouTube videos can be embedded using the <iframe> tag. 

<iframe width="560" height="315" 
src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
title="YouTube video" 
frameborder="0" 
allowfullscreen></iframe>

allowfullscreen: lets video play in full-screen mode.

<embed> Tag & Its Importance

The <embed> tag allows embedding external resources like PDFs, Flash, or audio/video files. 

<embed src="document.pdf" width="500" height="400">

Autoplay Feature

<video src="video/sample.mp4" autoplay muted></video>

Multi-Page Websites üóê

A multi-page website has multiple HTML pages linked together using navigation.

Example:

<!-- index.html -->
<header>
    <h1>My Website</h1>
    <nav>
        <a href="index.html">Home</a> |
        <a href="about.html">About</a> |
        <a href="contact.html">Contact</a>
    </nav>
</header>

<main>
    <h2>Welcome to Home Page</h2>
</main>

<footer>
    <p>&copy; 2025 My Website</p>
</footer>

Combining iframes with Tables

<table border="1">
    <tr>
        <td><iframe src="about.html" width="300" height="200"></iframe></td>
        <td><iframe src="contact.html" width="300" height="200"></iframe></td>
    </tr>
</table>

Deploying a Website on Netlify

Netlify is a free hosting service for static websites. You can easily deploy your HTML project.

  1. Go to Netlify Website.
  2. Sign up using GitHub/Google.
  3. Create a new site ‚Üí Upload your HTML project folder.
  4. Click on "Deploy Site".
  5. Your project will be live on a Netlify domain.
