Understanding the foundation of every web page

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design webpages. HTML structures the content of a website using elements like headings, paragraphs, images, and links.

World First Website üìÑ

The world's first website was created by Tim Berners-Lee in 1991. It was hosted on a NeXT computer at CERN and explained the basics of the World Wide Web.

Visit the First Website

The Concept of Hyperlinks

Hyperlinks connect one page to another. They are created using the <a> tag.

Example:

<a href="https://www.example.com">Visit Example</a>

What is a Markup Language?

A markup language uses tags to "mark up" text in a document. HTML is a markup language, not a programming language.

Installing & Setting Up a Code Editor

Popular code editors for HTML include Visual Studio Code, Sublime Text, and Notepad++.

Creating Your First Webpage üìù

<!DOCTYPE html>
<html>
<head>
  <title>My First Webpage</title>
</head>
<body>
  <h1>Hello World!</h1>
  <p>This is my first webpage.</p>
</body>
</html>

Understanding Tags

HTML tags are keywords enclosed in <>. Example: <p>This is a paragraph</p>

What is Heading?

Headings are used to define titles or subtitles in a webpage. They range from <h1> (largest) to <h6> (smallest). 

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Smaller Heading</h3>

The Paragraph Tag

The <p> tag defines a paragraph.

<p>This is a paragraph example.</p>

What is an Element?

An element consists of an opening tag, content, and a closing tag. Example: <p>This is content</p>

Horizontal Rule (hr)

The <hr> tag creates a horizontal line.

Line Break (br)

The <br> tag is used to insert a single line break.

Introduction to HTML Lists

Lists help organize information. HTML supports two main types:

Ordered List (numbered)

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Unordered List (bulleted)

<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>

Creating Hyperlinks

<a href="https://google.com">Go to Google</a>

Adding Comments in HTML

Comments are notes for developers and are ignored by the browser.

<!-- This is a comment -->

The Anchor Tag (a)

Used to create links. It supports attributes like href, target.

Attributes Explained

Attributes provide additional information about elements. Example: <img src="image.jpg" alt="Description">

Adding Images (img)

The <img> tag is used to embed images in a webpage. 

<img src="photo.jpg" alt="My Photo" width="300">
