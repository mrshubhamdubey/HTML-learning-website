"Why" of Forms: Understanding the fundamental purpose of forms from first principles

What is an HTML Form? üìã

An HTML form is used to collect user input. For example, login forms, signup forms, search bars, and feedback forms. Forms are the backbone of interactive websites.

<form> Tag

The <form> tag defines a form in HTML. It acts like a container for input elements. 

<form action="/submit" method="post">
    
</form>

<input> Tag

The <input> tag is used to take input from the user. It has many types like text, password, email, etc. 

<input type="text" placeholder="Enter your name">

<label> Tag

The <label> tag is used to associate text with an input field for better accessibility. 

<label for="username">Username:</label>
<input type="text" id="username" name="username">

Detailed Explanation of Form Structure

<form action="/submit" method="post">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    <br><br>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    <br><br>

    <button type="submit">Login</button>
</form>

Name & Value Attributes

name: Attribute specifies the key for the input field.
value: Attribute specifies the default or selected value.

Placeholder Attribute

Placeholder is a hint shown inside the input field until the user types something. 

<input type="text" placeholder="Enter your name">

Other Input Types

Date: <input type="date"> <br><br>
File Upload: <input type="file"> <br><br>
Email: <input type="email" placeholder="example@mail.com"> <br><br>
Phone: <input type="tel" placeholder="123-456-7890">

<textarea> Tag

The <textarea> tag is used for multi-line text input. 

<textarea rows="5" cols="40" placeholder="Write your message here..."></textarea>

<select> and <option> Tag

Dropdown menus are created using the <select> tag. 

<select name="city">
    <option value="delhi">Delhi</option>
    <option value="mumbai">Mumbai</option>
    <option value="kolkata">Kolkata</option>
</select>

The <button> tag is used to create clickable buttons. 

<button type="submit">Submit</button>
<button type="reset">Reset</button>
